# FilmsVoiceOver
Creating Voice Over for any film from subtitles (voice creating by Ivona, using DonationAlerts API)

Warning: this script was written for voice over on Russian, from concretised subtitles and video format.

Переводит всё неплохо, но нужно много что настраивать под себя, если вы хотите использовать этот скрипт) 

Чтобы можно было что-то переводить, зарегистрируйтесь в DonationAlerts, получите ключ и укажите его define в SubsParser.php
